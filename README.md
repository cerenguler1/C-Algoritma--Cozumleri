Bu repo, algoritma mantığımı geliştirmek için Codewars üzerinde çözdüğüm problemleri içerir.

Çözülen Problemler: 1. Sums of Consecutive Integers: Verilen matematiksel formüle göre ardışık sayıların konumunu bulan algoritma. 2. Sum of a Beach: Bir string içinde (büyük/küçük harf fark etmeksizin) belirli anahtar kelimelerin kaç kez geçtiğini bulan algoritma.
